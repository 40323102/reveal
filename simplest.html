做法:

1.下載fix和reveal 2.開fix和start 開始在cmd輸入指令 3.git clone網址然後建分支 4.然後再複製revaeal裡的東西除了read me 貼到自己的資料夾 5.在git add -A所有的檔案拍快照 6.git commit -m "initial add all files" 解開 7.在確認自己身分git config 8.最後在git push origin gh-pages 9.之後在leo中更改內容案save然後再點pelican-content 10.在push一次
